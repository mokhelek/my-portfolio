## üîß INSTALLATION

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/extreme-brutalist-portfolio.git
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run migrations**
   ```bash
   python manage.py migrate
   ```

4. **Start the server**
   ```bash
   python manage.py runserver
   ```
